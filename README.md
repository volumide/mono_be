# Mono-Accessment

Clone the repository

##### run the command :

```
npm install
```

After installing all packages

##### run the command :

```
npx tsc -w
```

After rename .env.example fileo .env and update your .env file

##### run the command :

```
npx run db
```

##### run the command :

```
npx run start-dev
```
